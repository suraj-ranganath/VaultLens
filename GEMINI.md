# My Vault

This repo is a local-first markdown vault for durable personal knowledge and retrieval.

Read `AGENTS.md` for the full rules. Use `hot.md` first, `index.md` second, and `dashboards/` before crawling individual notes.

Key conventions:

- `raw/` and `imports/` are immutable source layers.
- `items/` contains canonical notes.
- `topics/`, `projects/`, and `outputs/` hold synthesis and reusable context.
- `log.md` is append-only.
- `hot.md` is a small rewritten cache, not a journal.
