# VaultLens: Windsurf Rules

This project is a local-first markdown vault for personal knowledge retrieval and maintenance.

Read `AGENTS.md` first. Then:

1. Read `hot.md` if it exists.
2. Use `index.md` and `dashboards/` as the primary navigation surfaces.
3. Treat `raw/` and `imports/` as immutable source layers.
4. Keep latest shared material first in generated views.
5. Keep `log.md` append-only and `hot.md` rewritten.
